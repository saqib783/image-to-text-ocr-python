# 🎯 Image-to-Text OCR Extractor in Python

A terminal-based **Optical Character Recognition (OCR) Tool** built using Python. Paste the path of any image, and the script instantly reads and extracts the text into a clean document.

## 🚀 Features
* **Interactive Gameplay:** Run directly inside your command line or VS Code terminal.
* **Path Auto-Cleaning:** Cleans quotes and slashes automatically when you drag and drop images.
* **Instant Feedback:** Processes your image immediately and updates you on the progress.
* **Final Summary:** Extracts paragraphs accurately and saves the final result to a `text.txt` file.

## 🛠️ Built With (Libraries & Tools)
* **Python 3.x** - Core programming language.
* **`easyocr` (External Library)** - Ready-to-use OCR with 80+ supported languages.
* **`torch` & `torchvision`** - Machine learning backend used by EasyOCR.

## 💻 How to Clone and Run the Project

Follow these simple steps to download and run the tool on your local machine:

### 1. Clone the Repository
Open your terminal or command prompt and run the following command to download the code:
```bash
git clone https://github.com/saqib783/image-to-text-ocr-python.git
```

### 2. Navigate to the Directory
Go into the project folder:
```bash
cd image-to-text-ocr-python
```

### 3. Install Dependencies
Install the required OCR package before running the script:
```bash
pip install easyocr
```

### 4. Run the Script
Execute the Python file using your terminal:
```bash
python ocr.py
```

---

## 🎮 How to Play
1. Once the program starts, you will see a prompt in the terminal: `paste path here`.
2. Drag and drop your image file or paste its path.
3. The script will instantly read the image:
   * It processes lines using advanced paragraph detection.
   * It automatically creates a `text.txt` file in the same directory.
4. At the end, the system displays: `we are done` 🎉
